# Reminder

## Project
- macOS desktop app, AppKit, no Storyboard
- Entry point: main.swift (manual NSApplication launch)
- VIPER architecture
- Sandboxed (entitlements)

## Dependencies
- DrumrollTimePicker from https://github.com/OneEuro/DrumrollTimePicker.git

## Code style
- `self.` always explicit
- Use `guard` for early exit
- Imports sorted alphabetically
- 4-space indentation
